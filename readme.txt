вставьте свой токен в server.py, чтобы сервер заработал


api:
---
список методов:
auth
reg_user
get_mem
like_mem
skip_mem
---


==
метод auth:
позволяет проверить корректность данных пользователя
параметры: 
1. user_id - целое неотрицательное число, не длиннее 20 символов
2. user_pass - целое неотрицательное число, не длиннее 20 символов
ответ:
{
    'state': 'error'/'ok',
    'data': строка с описанием
}

пример:
http://localhost/auth?user_id=123&user_pass=123
==

==
метод reg_user:
позволяет зарегистрировать пользователя
параметры: 
1. user_id - целое неотрицательное число, не длиннее 20 символов
2. user_pass - целое неотрицательное число, не длиннее 20 символов
ответ:
{
    'state': 'error'/'ok',
    'data': {
        'valid': True/False,
        'msg': строка с описанием
        }
}

пример:
http://localhost/reg_user?user_id=123&user_pass=123
==

==
метод get_mem:
позволяет пользователю получить картинку
параметры: 
1. user_id - целое неотрицательное число, не длиннее 20 символов
2. user_pass - целое неотрицательное число, не длиннее 20 символов
ответ:
{
    'state': 'error'/'ok',
    'data': строка с описанием ошибки/объект mem
}

объект mem - это объект Фотография из vk api v=5.131, где
вместо ключа 'user_id' установлен ключ 'author'
mem['author'] = {
            'id': Фотография['user_id'],
            'first_name': имя автора фотографии,
            'last_name': фамилия автора фотографии
}

пример:
http://localhost/get_mem?user_id=123&user_pass=123
==

==
метод like_mem:
позволяет поставить лайк картинке от имени пользователя
параметры: 
1. mem_id - Идентификатор фотографии
2. user_id - целое неотрицательное число, не длиннее 20 символов
3. user_pass - целое неотрицательное число, не длиннее 20 символов
ответ:
{
    'state': 'error'/'ok',
    'data': строка с описанием
}

пример:
http://localhost/like_mem?mem_id=63453636&user_id=123&user_pass=123
==

==
метод skip_mem:
позволяет пользователю убрать картинку из ленты предложений
параметры: 
1. mem_id - Идентификатор фотографии
2. user_id - целое неотрицательное число, не длиннее 20 символов
3. user_pass - целое неотрицательное число, не длиннее 20 символов
ответ:
{
    'state': 'error'/'ok',
    'data': строка с описанием
}

пример:
http://localhost/skip_mem?mem_id=63453636&user_id=123&user_pass=123
==
